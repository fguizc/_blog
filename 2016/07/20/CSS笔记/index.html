    ## 什么是CSS

    CSS是**Cascading Style Sheets即为层叠样式表**
    CSS分为:

    • 外部样式(通过在HTML文件中确定一个外来的独立的CSS样式表)
    • 内联样式(通过在HTML当前文件中写入CSS样式)
    • 内部样式(通过在HTML标签结构内写入CSS样式)

    其中优先级为 外部样式?内联样式<内部样式
    !!!其中外部样式还是内部样式优先是根据就近原则

    CSS语法

    CSS由多个规则组成,每个规则由**选择器(selector)** 、属性(property) 、值(value) 组成。

    selectorlist { property: value; [more property:value; pairs] }

    选择器（Selector）：多个选择器可以半角逗号（,）隔开。
    属性（property）：CSS1、CSS2、CSS3规定了许多的属性，目的在控制选择器的样式。
    值（value）：指属性接受的设置值，多个关键字时大都以空格隔开。 属性和值之间用半角冒号（:）隔开，属性和值合称为“特性”。多个特性间用“;”隔开，最后用“{ }”括起来。

    选择器

    选择器种类

    • 类选择器
    • 标签选择器
    • ID选择器
    • 通用选择器
    • 属性选择器
    • 组合选择器
    • 伪类选择器

    类选择器(class)

    .elementname

    标签选择器

    elementname(h1/p/div)

    ID选择器

    #elementname

    通用选择器

    *

    属性选择器

    [attribute] / [attribute="value"] / [attribute~="value"] etc

    组合选择器

    A > B :选择A下一层的元素B
A ~ B :选择与A同父层的元素B
A + B :选择与A相邻的元素B
A B   :咋选A的后代元素B

    伪类

    a:link {color: #FF0000}		 未访问的链接 
a:visited {color: #00FF00}	 已访问的链接 
a:hover {color: #FF00FF}	 鼠标移动到链接上 
a:active {color: #0000FF}	 选定的链接 
:focus				向拥有键盘输入焦点的元素添加样式
:first-child		向元素的第一个子元素添加样式
:lang				向带有指定 lang 属性的元素添加样式	


	:hover需放在 :link :visited后 , :active放 :hover后
						即为LVHA

    伪元素

    :after
:before
:first-letter
:first-line
:selecton

    CSS选择器权重

    1. 尽可能不要使用 “!important”, !important会覆盖所有样式且没有结构与上下文
    2. 减少选择器的个数
    3. 相同权重以后出现的选择器为最后规则
    4. 不同的权重,权重高则生效

    如何计算优先级

    行内样式 		 +1000 
ID选择器		   +100
类选择器/伪类 	  +10
标签选择器/伪元素    +1

    缩写

    • 边框的缩写
      border :[(border-width)|(border-style)|(border-color)]

    • 文字的缩写
      font :[(font-weight)|(font-size)|/(line-height)|(font-family)]

    • 背景的缩写
      background :[(bg-color)|(bg-img)|(bg-repeat)|(bg-attachment)|(bg-position)] background-attachment 属性设置背景图像是否固定或者随着页面的其余部分滚动, value=scroll | fixed;
