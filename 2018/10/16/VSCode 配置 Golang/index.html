    • VSCode 配置 Golang

    由于一个众所周知的原因，并不能顺畅地在VSCode内安装完Go插件后自动安装其所需要的依赖。

    Windows

    在win10下安装完成Go后（没有更改默认配置，直接装到了C盘），可以找到系统配置并且自动完成了GoPath和GoRoot的配置。

    1. 命令行进入到GoPath对应路径，于src目录下创建golang.org/x路径
    2. 执行git clone [https://github.com/golang/tools.git](https://github.com/golang/tools.git) tools，将Golang在github上提供的镜像包clone到本地
    3. 在VSCode使用命令行进入到步骤2的路径，依次安装Go插件提示的缺失部分
    4. 其中golint没有直接包含在tools，需要额外在$GOPATH/src/golang.org/x下再下次一次git clone https://github.com/golang/lint.git
    5. 再次于命令行内进入GoPath执行go install golang.org/x/lint/golint

    事实上在我查询google的过程中，大多数推荐使用go get -u -v url去下载，然而即便挂上了全局代理也依旧失败。 go get本质是从给定的url下载或更新指定的包，并对他们进行编译和安装。因此曲折一点的办法就是先把这些包下载下来，再通过go install完成安装。
